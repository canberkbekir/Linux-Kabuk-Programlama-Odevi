# Linux Kabuk Programlama Ödevi
## _CV Yaratacı_


Projenin amacı bash script ile html formatında CV yaratıp temasını ayarlayabilme

## Özellikler

- Kendi bilgilerinizle CV yaratabilme
- Temasını seçebilme

## Çalıştırma
```sh
sudo ./first.sh
```
